# Security Policy

## Supported Versions

Only the latest version of the application is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v2.x    | :white_check_mark: |
| < 2.x   | :x:                |

## Reporting a Vulnerability

We take the security of the Bharat Heritage Portal seriously. If you find a security vulnerability, please do not report it publicly via GitHub Issues. 

Instead, please report vulnerabilities by contacting the maintainers directly or emailing [bvrao204@example.com](mailto:bvrao204@example.com).

Please include the following details in your report:
- Type of issue (e.g., cross-site scripting, authentication bypass, dependency flaw)
- Step-by-step instructions to reproduce the issue
- Potential impact of the vulnerability

We will acknowledge receipt of your report within 48 hours and work with you to coordinate a security patch before public disclosure.
